The Nu-Checker

What is Nu-Checker

The Nu HTML Checker is a little program that allows you to enter in the url of a HTML site that you are working on and it will check it for errors, and as they describe it on the website it self, "To catch unintended mistakes—mistakes you might have otherwise missed—so that you can fix them." Simply put, it is a program that will catch errors in your HTML code and tell you what the issues are so you can fix them yourself. Similar to putting your essay through one of those grammar or plagiarism checkers before you turn it in.

How do I use it?

  • Go to the Nu Checker website
  • Copy the URL of your HTML file/website into the box
  • It will then display the issues that your file contains so you can fix them
  • Fix the issues accordingly, and repeat the steps again to see if there are any remaining issues with the file.
Tip: after checking the HTML file in the checker, the URL of the Nu Checker site will change, if you want to show anybody that there are no issues with you site, send them this link!
