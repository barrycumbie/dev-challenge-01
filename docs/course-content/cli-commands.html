Command Line Commands

  • Linux Terminal Commands

    • pwd
      • Returns your current directory location
    • ls <optional-flag>
      • Shows the contents of your current directory
      • Useful flags:
        • -l (Lists all files with information about each file)
        • -a (Lists all files including hidden files)
        • -la -al (These flags can be used together)
    • cd <directory-name>/
      • Change Directory
      • cd .. takes you to the parent folder
      • Note: The whack (/) is not always needed but helps for clarity to see that this is a folder
    • mkdir <directory-name>/
      • Creates a new directory
      • Note: The whack (/) is not always needed but helps for clarity to see that this is a folder
    • rm <file-name>
      • Deletes a file

  • Windows Command Line Commands

    • chdir
      • Returns your current directory location
    • dir <optional-flag>
      • Shows the contents of your current directory
      • Useful flags:
        • /O:SortOrder
          • N : By name (alphabetic)
          • S : By size (smallest first)
          • E : By extension (alphabetic)
          • D : By date/time (oldest first)
          • G : Group directories first
          • - : Prefix to reverse the order
        • /Q Displays the owner of the file
      • Note: dir shows hidden files by default
    • cd <directory-name>/
      • Change Directory
      • cd .. takes you to the parent folder
      • Note: The whack (/) is not always needed but helps for clarity to see that this is a folder
    • md <directory-name>/
      • Creates a new directory
      • Note: The whack (/) is not always needed but helps for clarity to see that this is a folder
    • del <file-name>
      • Deletes a file
