Command Line Commands

  • Linux Terminal Commands

    • pwd
      • Returns your current directory location
    • ls
      • Shows the contents of your current directory
    • cd
      • Change Directory
      • cd .. takes you to the parent folder
    • mkdir
      • Creates a new directory
    • rm
      • Deletes a file

  • Windows Command Line Commands

    • chdir
      • Returns your current directory location
    • dir
      • Shows the contents of your current directory
    • cd
      • Change Directory
      • cd .. takes you to the parent folder
    • md
      • Creates a new directory
    • del
      • Deletes a file
