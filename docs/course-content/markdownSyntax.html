Helpful markdown syntax

Lists: To make ordered lists you just put a number, like 1., with the period before your sentence. It doesn't matter what number you type because it will order it for you.

Unordered lists can be made by using an asterisk *, minus -, or plus + sign before your text.

Code syntax - in case you have questions on code. After the three ticks state the language for syntax highlighting.
```javascript
var s = "JavaScript syntax highlighting";
alert(s);
```
You can also use inline `code` like this with `back-ticks around` it.
